# 🚀 SmartCareerForge

### AI-Powered Career Guidance & Skill Recommendation Platform

**SmartCareerForge** is an AI-driven career guidance platform designed to help students and aspiring professionals discover suitable career paths based on their interests, skills, and queries.

The system provides **intelligent role suggestions, career insights, and guidance** through an interactive web interface built with Python and Streamlit.

This project was developed as part of a **Final Year Software Development Project (SDP)**.

---

# 📌 Project Overview

Choosing the right career path can be challenging, especially for students who are unsure about available roles or required skills. SmartCareerForge aims to simplify this process by providing an AI-assisted platform that analyzes user inputs and suggests relevant career roles along with guidance.

The platform allows users to:

* Explore career options
* Understand skill requirements
* Ask career-related questions
* Receive AI-generated recommendations

---

# ✨ Key Features

### 🎯 Career Role Recommendations

Suggests suitable career roles based on user inputs and interests.

### 🤖 AI Career Guidance Assistant

Helps users who are unsure about which career path to choose.

### 📊 Interactive Visualizations

Displays insights and results through graphical visualizations.

### 💬 Query-Based Career Assistance

Users can ask questions about roles, skills, and career paths.

Built with Streamlit for an intuitive and interactive web experience.

# ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/arigasaicharanreddy/smartcareerforge.git
```

### 2️⃣ Navigate to the project directory

```bash
cd smartcareerforge
```

### 3️⃣ Create a virtual environment (optional but recommended)

```bash
python -m venv venv
```

Activate the environment

Windows

```bash
venv\Scripts\activate
```

Linux / Mac

```bash
source venv/bin/activate
```

---

### 4️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

---

### 5️⃣ Run the application

```bash
streamlit run app.py
```

The application will start and open in your browser.
